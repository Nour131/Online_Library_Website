# 📚 Online Library Website

An online library management website developed using Django that allows users to browse, search, and manage books through an interactive web interface.

---

## Project Overview

This project is a web-based online library system designed to simplify book management and improve user interaction with digital library services. The website provides functionalities for browsing books, managing library records, and handling user interactions efficiently.

---

## Features

- User-friendly web interface
- Book browsing and searching
- Add, update, and delete books
- User authentication system
- Database management using Django ORM
- Responsive website design
- Organized book catalog system

---

## Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite
- Bootstrap

---

## Project Structure

The project follows the Django MVC/MVT architecture and includes:

- Frontend templates
- Backend logic using Django views
- Database models
- Static files and styling
- User authentication and routing

---

## Main Functionalities

### User Features
- Browse available books
- Search for books by title or category
- View book details
- Send contact messages to admin

### Admin Features
- Manage books and library records
- Add new books
- Update existing book information
- Delete books from the system
- View users' messages
